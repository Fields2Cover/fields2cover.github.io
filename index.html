Fields2Cover

Robust and efficient coverage paths for autonomous agricultural vehicles

The github repository is here.

The Coverage Path Planning problem (CPP) aims to create a path to cover an area with one or several vehicles. Although the development of this project is focused on offline planning of agricultural vehicles, the library accepts pull requests from other types of coverage planners.

Contribute

If you find any issue/bug/proposal, open an issue and we will try to solve/discuss it.

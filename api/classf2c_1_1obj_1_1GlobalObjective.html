class f2c::obj::GlobalObjective

Base class for global objectives.

Global objectives are those objectives that affect the planning of the field. Some of the could be f2c::obj::NSwath or f2c::obj::FieldCoverage

Subclassed by f2c::obj::FieldCoverage, f2c::obj::NSwath, f2c::obj::Overlaps, f2c::obj::SwathLength
